# Bnyan (بنيان) — Premium Real Estate Investment Platform

![Bnyan Logo](assets/images/logo.png)

Bnyan is a high-end, responsive real estate investment platform designed to bridge the gap between savvy investors and premium real estate developers in Saudi Arabia. The platform emphasizes transparency, ease of use, and a sophisticated aesthetic.

## 🌟 Key Features

- **Premium UI/UX**: Built with a custom design system featuring **Cairo** typography, smooth transitions, glassmorphism accents, and a high-contrast color palette.
- **Full Responsiveness**: Optimized for every device size, from ultra-wide desktops to small smartphones (320px+).
- **Interactive Investment Flow**: A seamless 4-step process for discovering projects and confirming investments.
- **Investor Dashboard**: A clean interface for tracking active investments, returns, and project updates.
- **Developer Portal**: A dedicated space for developers to manage projects and post updates for investors.
- **RTL Optimized**: Designed from the ground up for Arabic-speaking users with perfect Right-to-Left alignment.

## 📂 Project Structure

The project consists of 6 core pages:

1.  **[index.html](index.html)**: The landing page highlighting the value proposition, live stats, and featured projects.
2.  **[page2-projects.html](page2-projects.html)**: A project discovery hub with advanced filtering (City, Price, Returns).
3.  **[page3-detail.html](page3-detail.html)**: Deep-dive into specific projects with galleries, financial KPIs, and investment timelines.
4.  **[page4-invest.html](page4-invest.html)**: A focused investment confirmation page with a real-time ROI calculator.
5.  **[page5-dashboard.html](page5-dashboard.html)**: Personalized investor dashboard for tracking portfolio performance.
6.  **[page6-developer.html](page6-developer.html)**: The developer's management console for project updates and investor communication.

## 🛠 Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3 (Vanilla)**: Custom styling without dependencies.
- **JavaScript**: Interactive elements, dynamic calculations, and UI state management.
- **Google Fonts**: [Cairo](https://fonts.google.com/specimen/Cairo) for a modern, professional Arabic look.

## 📱 Responsive Breakpoints

The platform utilizes a multi-tier responsive system:
- **Desktop (>1200px)**: Full multi-column grids and expanded navigation.
- **Tablet (1024px - 900px)**: Refined spacing and balanced column ratios.
- **Mobile Collapse (900px)**: Navigation links hide into a clean mobile-first view.
- **Smartphone (640px)**: Grid stacking for single-column readability.
- **Ultra-Mobile (<400px)**: Optimized typography and full-width touch targets for small handsets.

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Explore the seamless flow from discovery to investment!
